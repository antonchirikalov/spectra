# Проект «МойГараж» — краткий бриф

## Контекст
Сеть автосервисов «МойГараж» (12 филиалов в Москве и области) хочет мобильное приложение
для клиентов и внутренний портал для приёмщиков заказов.

## Цели
- Клиент записывается на сервис онлайн: выбор филиала, услуги, даты и мастера.
- Клиент видит статус ремонта в реальном времени (принят / в работе / готов).
- Push-уведомления о готовности автомобиля.
- Приёмщики работают с очередью заказов через веб-портал.

## Функциональные пожелания
- Регистрация по номеру телефона (SMS-код).
- История обслуживания автомобиля (по VIN).
- Оплата онлайн (карта, СБП) или при получении.
- Интеграция с 1С:Управление автосервисом — заказы, склад запчастей, клиенты.

## Ограничения
- Срок пилота: 2 филиала, 3 месяца.
- Приложение: iOS и Android из одной кодовой базы.
- Персональные данные — хранение в РФ (152-ФЗ).
